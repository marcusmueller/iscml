
NAME
    upload - upload a client scope element

SYNOPSIS
    upload element-name

DESCRIPTION
    This command uploads a client scope element to the server. The element
    must first be defined in the top level templates/client_elements 
    directory. Once a client scope element is uploaded to the server, 
    jobs that reference that element can be launched. See UserGuide.txt 
    for a description of how to create define client elements.
